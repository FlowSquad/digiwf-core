# Stack

For local development you can use the following tools:

- **Event Bus**: Kafka (and zookeeper)
- **S3 Storage**: [Minio](https://min.io/docs/minio/linux/index.html)
- **Email Server**: [Mailhog](https://github.com/mailhog/MailHog)

## Docker

```bash
docker compose up -d
```
